La funció callback és una funció que es passa a una altra de manera semblant a un argument. Després és invocada per una funció externa per completar una acció. Exemple: //La funció fnNom és el callback, que conté el nom d'una persona a l'hora d'iniciar sessió. function fnNom(nom) { console.log("Hola, sóc "+nom); } //La funció demanaNom fa que la pàgina web pregunti el nom de l'usuari i el nom s'enregistrarà a fnNom. function demanaNom(callback) { let nom = prompt("Hola. Com et dius?"); callback(nom); } //S'activa la funció demanaNom amb fnNom com el seu paràmetre. D'aquesta manera, demanaNom pot accedir al paràmetre de fnNom i mostrar-lo. demanaNom(fnNom); 7.
